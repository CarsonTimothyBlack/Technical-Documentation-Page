1/23/2024 - - JS Variable Basics

Overview

What is a Variable? A Variable is a container for storing data.

1. Declaring a variable

To use a variable, you must first “declare” it. For our class, we will be using the most common way of declaring a variable, namely using “var”. Here's an example of a variable being declared

var fishName = “Wanda”

You can then reference that variable:

console.log(“My fish's name is” + fishName);

The above code will output “My fish's name is Wanda”.

You may also assign a different value to an existing variable.

var fishName = “Wanda”;

fishName = “Freddy”;

console.log(“My fish's name is” + fishName);

The above code will output “My fish's name is Freddy.

2. Naming a variable

Make sure to make your variable names descriptive, so that human readers (like you later on) can understand your code better.

GOOD

var freezingTempurature = 32;

BAD

var t = 32;

The most common variable naming format is called “camel case”. The variable name starts in lower case, then any added word has its first letter capitalized. This creates a "humped" effect...like a camel!

var thisVariableIsNamedUsingCamelCase = true;

3. Data types and structures that can be stored in a variable

Strings

A string value is anything with quotes around it. Even a number with quotes around it is technically a string

var myComputerTalks = “Hello World”;

var shout = “AHHHHH!”;

var excuse = “My dog ate my homework.”;

var notReallyNumber = “8”;

var iAmEightySeven = “8” + "7";

Numbers

A number value is, well, any number. Some coding languages require you to specify what kind of number you are declaring, like a whole number, or one with a decimal, or a negative number. JavaScript doesn't require you to be that specific. You just use any number for your value (without quotes), and JS will know it's a number.

var favoriteNumber = 13;

var holyDecimal = 7.77;

var negativeNancy = -13;

var four = 2 + 2;

Boolean

A boolean value is either true or false (again without quotes)

var godIsGood = true;

var iAmPerfect = false;

Array

An array is a data structure that is a set of values. This set is surrounded by square brackets “[]” and each item is separated by a comma. These can be a mix of data types, if needed.

var superHeroStrings =[“Batman”, “Spider-Man”, “Dogman”, “Person-Man”];

var luckyNumbers = [7, 13, 21, 88, 34];

var mixedDataTypeArray = [“A string”, 4, true, “I'm the last item!”];

Object

An object is another data structure that consists of what are called “key” and “value” pairs. This structure is surrounded by curly brackets “{}” and, again, each item pair is separated by a comma. Again, these items can be a mix of data types, if needed.

var myDog ={
name: “Lola”,
age: 2,
lovesBath: false,
favoriteWord: “Bark”
};

1/30/2024 - - JS Variable Methods and Operators

1. Operators

“Operators” is the coding word to describe things like addition, subtraction, assignment, etc. For example, we use our Assignment Operator, or “=”, every time we give a variable a value:

var thisVariableEqualsTwenty = 20;

You are probably familiar with how most of these operators will work, but I'm going to give you examples of most of the ones you will come across. See if you can guess what the outcome/output of each variable example will be:


Addition Operator +

var whatIsMyValue = 1 + 3;

console.log(whatIsMyValue);;


var whatIsMyValueNow = “1” + “3”;

console.log(whatIsMyValueNow);


var whatDoISay = “I love” + “ Pizza” + “!”;

console.log(whatDoISay);


var whatWillThisArrayDo = [“Potato”, “Carrot”, “Beet”] + [1, 8, 20];

console.log(whatWillThisArrayDo);

Increment Operator ++

var currentCountValue = 1;

currentCountValue++;

currentCountValue++;

console.log(currentCountValue);

Subtraction Operator -

var whatIsMyValue = 4 - 3;

console.log(whatIsMyValue);


var whatIsMyValueNow = “4” - “3”;

console.log(whatIsMyValueNow);

JS converts strings to numbers, if it makes sense to do so.


var whatDoISay = “I love” - “ Pizza”;

console.log(whatDoISay);

JS can't really subtract any other kind of string from another.

…At least in not so straight forward a way as with “-”


var whatWillThisArrayDo = [“Potato”, “Carrot”, “Beet”] - [1, 8, 20];

console.log(whatWillThisArrayDo);

Same with arrays. You'll get an error with the above code.

Decrement Operator --

var currentCountValue = 5;

currentCountValue--;

currentCountValue--;

console.log(currentCountValue);

Multiplication Operator *

var whatIsMyValue = 4 * 3;

console.log(whatIsMyValue);

Exponentiation Operator ** (to the power of)

var whatIsMyValue = 3**2; // 3 squared

console.log(whatIsMyValue);


var whatIsMyValueNow = 2**4; // 2 to the power of 4

console.log(whatIsMyValueNow);

Division Operator /

var whatIsMyValue = 10/2;

console.log(whatIsMyValue);


var whatIsMyValueNow = 5/3;

console.log(whatIsMyValueNow);

Modulus Operator %

var whatIsMyValue = 5%2;

console.log(whatIsMyValue);


var isNumberDivisibleByThree = 12%3;

console.log(isNumberDivisibleByThree);

Check to see if a number is divisible by another number by seeing if modulus operator returns a 0.

2. Methods

“Methods” are functions you can run on a particular variable. These are functions that are built into JavaScript, and are great ways to test or manipulate your variable. The basic syntax is variableName.methodName. Often the method name is followed by parentheses. On a side note, you can build your own methods in JavaScript. More on that in later weeks.

Below are some examples of methods:

String Methods

var longString = “Why am I so long?”;

console.log(longString.length);


var findTheZ = “What index is Z at?”;

console.log(findTheZ.charAt(14));


var iCanGetBigger = “i will be big someday.”;

console.log(iCanGetBigger.toUpperCase());


var iWantToBeSmall = “CAN YOU PLEASE SHRINK ME?”;

console.log(iWantToBeSmall.toLowerCase());

Number Methods

var iWantToBeString = 567;

console.log(iWantToBeString.toString() + “ is now a string!”);

Array Methods

var favoriteFruits =[“apple”, “pear”, “tomato”, “joy”];

console.log(favoriteFruits.length);

console.log(favoriteFruits.join(“,”));


favoriteFruits.push(“kindness”);

console.log(favoriteFruits);


favoriteFruits.pop();

console.log(favoriteFruits);

2/6/2024 - - JS Arrays

Overview

What is an Array? An array is a special variable, which can hold more than one value.

1. Creating an Arrray

To create an array, use the following syntax:

var arrayName = [item1, item2, item3, …];

2. When to use an array

Arrays are a great way to structure your data. For instance, you could create 3 different food variables:

var food1 = “banana”;

var food2 = ham;

var food3 = taco;

And that would be fine, and work in JS. However, when you start to see similar data items, it may make more sense to group them into an array. Plus, there are some array-specific methods, or built in JS functions, that you couldn’t run on a single string value. In the example above, it may be more elegant in your code to have one food variable, with different items. Namely, a food array!

var foods = [“banana”, “ham”, “taco”];

3. Accessing array items

To access a value for a single item variable, like a String, Number, or Boolean, you simply reference the variable name:

var tinyString = “small”;

var sentenceStart = “Bugs are “;

console.log(sentenceStart + tinyString);

In the above example, we simply typed the variable names, in this case adding them together, to access their values.

With arrays, we would reference the array name, but also specify the index within the array for the value we wish to access. Array indexes start with 0, meaning the first item is at index 0, while the last item is at an index that equals the length of the array minus 1. Let me illustrate:

var kickArray = [“roundhouse”, “karate”, “scissor”, “punch?”];

console.log(kickArray[0]); // Outputs “roundhouse”

console.log(kickArrray.length); // Outputs 4

console.log(kickArray[kickArray.length - 1]); // Outputs “punch?”

The kickArray length is 4, since there are 4 items. But, since the index starts at 0, the last item, in this case, would be at index 3, or kickArray.length (which equals 4) - 1.

Just like with single value variables, you can change the value of an array item:

var oneOfTheseIsOdd = [“Apple”, “Orange”, “Pajamas”, “Mango”, “Peach”];

console.log(oneOfTheseIsOdd[2]); // Outputs “Pajamas”

oneOfTheseIsOdd[2] = “Pear”;

console.log(oneOfTheseIsOdd[2]); // Now outputs “Pear”

4. Array methods

There are several methods you can run on an array. One might even say that there are an array of array methods. Well, maybe just I would say that. Anyway, I'm only going to list the most commonly used ones below. But feel free to research more on W3schools.

Array.length

We've seen this one before! This tells you how many items are in the array.

var howLongAmI = [1, “test”, true, “confusing”, 88];

console.log(howLongAmI.length); // Outputs 5

Array.toString()

This method turns your array into one big string:

var iWantToBeAString = [“I”, “have”, 3, “kids”];

console.log(iWantToBeAString.toString); // “I, have, 3, kids”

Array.join()

This method turns your array into one big string, with each item separated by the character you provide in the parentheses:

var iWantToBeAString = [“I”, “have”, 3, “kids”];

console.log(iWantToBeAString.join(“ “)); // “I have 3 kids”

console.log(iWantToBeAString.join(“*“)); // “I*have*3*kids”

console.log(iWantToBeAString.join(“shm“)); // “Ishmhaveshm3shmkids”

Array.push()

This is the most common method to add an item to an array. It adds the specified item to the end of the array:

var iAmMissingSomething = [“I”, “have”, 3];

iAmMissingSomething.push(“baboons”);

console.log(iAmMissingSomething.join(“ “)); // “I have 3 baboons”

2/13/2024 - - JS Objects

Overview

What is an Object? An object is a variable that describes many parts of a single thing

1. Real life objects, properties, and methods

We experience objects in real life all the time that have properties (or descriptions), and usually methods (or actions) as well. For example, in real life you may have a…

OBJECT - - car


PROPERTIES - -

  • car.name = ol' Bessy
  • car.brand = Chrysler
  • car.color = red
  • car.age = old


METHODS - -

  • car.start()
  • car.drive()
  • car.break()
  • car.stop()
  • car.explode()

2. JavaScript Objects

You can translate all this into a JavaScript object that looks like this:

var myCar = {
 name: “ol Bessy”,
 brand: “Chrysler”,
 color: “red”,
 old: true,
 start: function() {
  return “Push break, and turn key”;
 }
 explode: function() {
  return “Keep driving this old thing. Eventually, it will explode.”;
 }
}

3. Basic Syntax

Below is the basic syntax for a JS object. As with most code, spaces and line breaks don;t often matter. This example is fine:

var objectName = {property1: value1, property2: value2, property3: value3};


You don't need to break lines for each property. However, line breaks do make it easier for humans to read it, and they become almost necessary, when you start to include methods. So, you will more often see objects with line breaks like this:

var objectName = {
 property1: value1,
 property2: value2,
 property3: value3
 methodName: function() {
  //code to run when method is called
 }
}

4. Accessing Object Properties and Methods

To access properties and methods in an Object, we first call the object name, then reference the property or method name, with a period in-between. Using our car object above we would access items like this:

console.log(car.name); // Outputs “ol Bessy”

console.log(car.old); // Outputs true

console.log(car.start()); // Outputs “Push break, and turn key”
