1/23/2024 - - JS Variable Basics

Overview

What is a Variable? A Variable is a container for storing data.

1. Declaring a variable

To use a variable, you must first “declare” it. For our class, we will be using the most common way of declaring a variable, namely using “var”. Here's an example of a variable being declared

var fishName = “Wanda”

You can then reference that variable:

console.log(“My fish's name is” + fishName);

The above code will output “My fish's name is Wanda”.

You may also assign a different value to an existing variable.

var fishName = “Wanda”;

fishName = “Freddy”;

console.log(“My fish's name is” + fishName);

The above code will output “My fish's name is Freddy.

2. Naming a variable

Make sure to make your variable names descriptive, so that human readers (like you later on) can understand your code better.

GOOD

var freezingTempurature = 32;

BAD

var t = 32;

The most common variable naming format is called “camel case”. The variable name starts in lower case, then any added word has its first letter capitalized. This creates a "humped" effect...like a camel!

var thisVariableIsNamedUsingCamelCase = true;

3. Data types and structures that can be stored in a variable

Strings

A string value is anything with quotes around it. Even a number with quotes around it is technically a string

var myComputerTalks = “Hello World”;

var shout = “AHHHHH!”;

var excuse = “My dog ate my homework.”;

var notReallyNumber = “8”;

var iAmEightySeven = “8” + "7";

Numbers

A number value is, well, any number. Some coding languages require you to specify what kind of number you are declaring, like a whole number, or one with a decimal, or a negative number. JavaScript doesn't require you to be that specific. You just use any number for your value (without quotes), and JS will know it's a number.

var favoriteNumber = 13;

var holyDecimal = 7.77;

var negativeNancy = -13;

var four = 2 + 2;

Boolean

A boolean value is either true or false (again without quotes)

var godIsGood = true;

var iAmPerfect = false;

Array

An array is a data structure that is a set of values. This set is surrounded by square brackets “[]” and each item is separated by a comma. These can be a mix of data types, if needed.

var superHeroStrings =[“Batman”, “Spider-Man”, “Dogman”, “Person-Man”];

var luckyNumbers = [7, 13, 21, 88, 34];

var mixedDataTypeArray = [“A string”, 4, true, “I'm the last item!”];

Object

An object is another data structure that consists of what are called “key” and “value” pairs. This structure is surrounded by curly brackets “{}” and, again, each item pair is separated by a comma. Again, these items can be a mix of data types, if needed.

var myDog ={
name: “Lola”,
age: 2,
lovesBath: false,
favoriteWord: “Bark”
};

2/6/2024 - - JS Arrays

Overview

What is an Array? An array is a special variable, which can hold more than one value.

1. Creating an Arrray

To create an array, use the following syntax:

var arrayName = [item1, item2, item3, …];

2. When to use an array

Arrays are a great way to structure your data. For instance, you could create 3 different food variables:

var food1 = “banana”;

var food2 = ham;

var food3 = taco;

And that would be fine, and work in JS. However, when you start to see similar data items, it may make more sense to group them into an array. Plus, there are some array-specific methods, or built in JS functions, that you couldn’t run on a single string value. In the example above, it may be more elegant in your code to have one food variable, with different items. Namely, a food array!

var foods = [“banana”, “ham”, “taco”];

3. Accessing array items

To access a value for a single item variable, like a String, Number, or Boolean, you simply reference the variable name:

var tinyString = “small”;

var sentenceStart = “Bugs are “;

console.log(sentenceStart + tinyString);

In the above example, we simply typed the variable names, in this case adding them together, to access their values.

With arrays, we would reference the array name, but also specify the index within the array for the value we wish to access. Array indexes start with 0, meaning the first item is at index 0, while the last item is at an index that equals the length of the array minus 1. Let me illustrate:

var kickArray = [“roundhouse”, “karate”, “scissor”, “punch?”];

console.log(kickArray[0]); // Outputs “roundhouse”

console.log(kickArrray.length); // Outputs 4

console.log(kickArray[kickArray.length - 1]); // Outputs “punch?”

The kickArray length is 4, since there are 4 items. But, since the index starts at 0, the last item, in this case, would be at index 3, or kickArray.length (which equals 4) - 1.

Just like with single value variables, you can change the value of an array item:

var oneOfTheseIsOdd = [“Apple”, “Orange”, “Pajamas”, “Mango”, “Peach”];

console.log(oneOfTheseIsOdd[2]); // Outputs “Pajamas”

oneOfTheseIsOdd[2] = “Pear”;

console.log(oneOfTheseIsOdd[2]); // Now outputs “Pear”

4. Array methods

There are several methods you can run on an array. One might even say that there are an array of array methods. Well, maybe just I would say that. Anyway, I'm only going to list the most commonly used ones below. But feel free to research more on W3schools.

Array.length

We've seen this one before! This tells you how many items are in the array.

var howLongAmI = [1, “test”, true, “confusing”, 88];

console.log(howLongAmI.length); // Outputs 5

Array.toString()

This method turns your array into one big string:

var iWantToBeAString = [“I”, “have”, 3, “kids”];

console.log(iWantToBeAString.toString); // “I, have, 3, kids”

Array.join()

This method turns your array into one big string, with each item separated by the character you provide in the parentheses:

var iWantToBeAString = [“I”, “have”, 3, “kids”];

console.log(iWantToBeAString.join(“ “)); // “I have 3 kids”

console.log(iWantToBeAString.join(“*“)); // “I*have*3*kids”

console.log(iWantToBeAString.join(“shm“)); // “Ishmhaveshm3shmkids”

Array.push()

This is the most common method to add an item to an array. It adds the specified item to the end of the array:

var iAmMissingSomething = [“I”, “have”, 3];

iAmMissingSomething.push(“baboons”);

console.log(iAmMissingSomething.join(“ “)); // “I have 3 baboons”

2/13/2024 - - JS Objects

Overview

What is an Object? An object is a variable that describes many parts of a single thing

1. Real life objects, properties, and methods

We experience objects in real life all the time that have properties (or descriptions), and usually methods (or actions) as well. For example, in real life you may have a…

OBJECT - - car


PROPERTIES - -

  • car.name = ol' Bessy
  • car.brand = Chrysler
  • car.color = red
  • car.age = old


METHODS - -

  • car.start()
  • car.drive()
  • car.break()
  • car.stop()
  • car.explode()

2. JavaScript Objects

You can translate all this into a JavaScript object that looks like this:

var myCar = {
 name: “ol Bessy”,
 brand: “Chrysler”,
 color: “red”,
 old: true,
 start: function() {
  return “Push break, and turn key”;
 }
 explode: function() {
  return “Keep driving this old thing. Eventually, it will explode.”;
 }
}

3. Basic Syntax

Below is the basic syntax for a JS object. As with most code, spaces and line breaks don;t often matter. This example is fine:

var objectName = {property1: value1, property2: value2, property3: value3};


You don't need to break lines for each property. However, line breaks do make it easier for humans to read it, and they become almost necessary, when you start to include methods. So, you will more often see objects with line breaks like this:

var objectName = {
 property1: value1,
 property2: value2,
 property3: value3
 methodName: function() {
  //code to run when method is called
 }
}

4. Accessing Object Properties and Methods

To access properties and methods in an Object, we first call the object name, then reference the property or method name, with a period in-between. Using our car object above we would access items like this:

console.log(car.name); // Outputs “ol Bessy”

console.log(car.old); // Outputs true

console.log(car.start()); // Outputs “Push break, and turn key”

2/27/2024 - - JS Functions

Overview

What is a Function? A function is a block of code designed to perform a particular task.

1. Function Syntax

To create an array, use the following syntax:

function myFirstFunction() {
 // Put your block of code in here. It can be as short or long
 // as you like!
}

2. Function Example

Functions usually “return” something at the end. Here's an example of a function:

function itsAlive() {
 return “Hello World!”;
}


console.log(itsAlive());

3. Function Parameters

It is very common for functions to take parameters, also called “arguments”. These are variables you place inside the parenthesis, that can change depending on what you feed to the function, when calling it. Here's an example:

function myFavoriteColor(color) {
 return “My favorite color is ” + color;
}


console.log(myFavoriteColor(“green”));

console.log(myFavoriteColor(“blue”));

console.log(myFavoriteColor(2));

Make your function more flexible with more parameters:

function identifyMe(name, age, favoriteMovie) {
 return “My name is ” + name + “. I am “ + age + “years old. My favorite movie is “ + favoriteMovie + “.”;
}


console.log(identifyMe(“Burt”, 35, ”Waterworld”));

console.log(identifyMe(“Nancy”, 13, ”Brave Heart”));

4. Functions and Algorithms

Basically, functions and algorithms are the same thing. They both are a block of code that is meant to complete a particular task.

Here's an example of an algorithm (or function) that adds any two numbers given to it, when called:

function sumOfNumbers(num1, num2) {
 return num1 + num2;
}


console.log(sumOfNumbers(5, 8));

3/5/2024 - - JS Conditionals

Overview

What is a Conditional? A conditional is a block of code that only runs, if a condition is meant.

1. Conditional Syntax

To create a conditionall, use the following syntax:

if (condition to meet) {
 code to run, if condition is met
}


Here's an example of a real conditional:

if (5 < 6) {
 console.log("Yep! 5 is less than 6.");
}

2. if..else and else if

If your initial condition is not met, run a different block of code with "else":

var myMathScore = 90;
if (myMathScore > 80) {
 console.log(“I got an A!”);
} else {
 console.log(“I got kicked out of school!”);
}


If you want to check against several conditions, use "else if":

var myMathScore = 75;
if (myMathScore > 80) {
 console.log(“I got an A!”);
} else if (myMathScore > 70) {
 console.log(“I got a B!”);
} else if (myMathScore > 60) {
 console.log("I got a C...");
} else if (myMathScore > 50) {
 console.log("I got a D...");
} else if (myMathScore > 40) {
 console.log("I'm not doing so well...");
} else {
 console.log("They kicked me out of school!");
}

3. AND(&&)…OR(||)

If you want to check against multiple conditionals before a single block of code runs, you can use AND/OR syntax.


AND is used when every condition must be true to run a block of code:

if (5 > 6 && 11 > 2 && 7 == 7) {
 console.log(“Hooray! We can trust math!”);
}


OR is used if at least one condition is true:

if (5 > 6 || 11 > 55 || 7 == 2) {
 console.log(“...Some of that math is wrong. But...ok!”);
}

3/12/2024 - - JS Loops

Overview

What is a Loop? A loop is a way to run the same code over and over again, with a different value.

1. "for" Loop Syntax

To create a loop, use the following syntax:

for(var i = 0; i < 100; i++) {
 console.log(i);
}


The above loop will log the numbers 0 - 99 in the console.

Let's break down the parameters that lie within the parentheses:

  1. Create a counting variable and set its beginning value; var i = 0. This can be called anything, but usually it's “i”, since it often refers to the index of a particular set of values you are iterating over.
  2. Set the condition “i” must meet, in order to run the code block; i < 100. This translates in English to, “if the variable “i” is less than 100, run the loop code block again. The “i” variable will change each time the loop runs, and thus get tested against this condition before running each loop. It changes because of…
  3. Increase or decrease the count variable each time the loop runs; i++. i++ means that, everytime the loop runs, i will increase by 1.

2. Loop Examples

This loop will output 0 - 98 in console, counting by 2s, ie. 0, 2, 4, 6, etc.

for (var i = 0; i < 100; i + 2) {
 console.log(i);
}


This loop will output 100 - 1, counting backwards

for (var i = 100; i > 0; i--) {
 console.log(i);
}


This loop only outputs numbers that are divisible by 3

for (var i = 0; i < 100; i++) {
 if (i % 3 == 0) {
  console.log(i + “ is divisible by 3”);
 } }


This loop iterates through an array and outputs the array items

var foodArray = ["salmon", "pizza", "burger", "taco", "potato"]

for(var i = 0; i < foodArray.length; i++) {
 console.log(foodArray[i]);
}


Remember how we access array items by declaring the array name, then designating the index of the item between square brackets? For example, foodArray[2] would return "burger". In the above loop, we access a new array item each time the loop runs, because the "i" variable keeps increasing by 1. The first time the loop runs, i = 0, so we would be accessing the foodArray item at index 0, or foodArray[0].

Then i increases by 1. So next time through the loop, i = 1, and we access item at foodArray[1]. Then foodArray[2], then foodArray[3], and so on, until i is no longer less than foodArray.length.

foodArray.length would equal 5, since there are 5 items in the array. So, once i = 5, the loop stops, because 5 is no longer less than 5.

3/19/2024 - - JS Algorithms

Overview

What is an Algorithm? An algorithm is a set of commands to complete a specific task...Kind of like a function. Actually, exactly like a function.

Below are some Algorithm examples.

FizzBuzz

Write an algorithm that iterates through 1 - 100. If a number is divisible by 3, output the word “Fizz”, if divisible by 5, output the word “Buzz”, if divisible by both 3 and 5, output the word “FizzBuzz”. Otherwise, simply output the number.

Solution

for(var i = 1; i < 101; i++) {
 if (i % 15 == 0) { console.log(“FizzBuzz”); }
 else if (i % 3 == 0) { console.log(“Fizz”); }
 else if (i % 5 == 0) { console.log(“Buzz”); }
 else { console.log(i); }
}

Word Reverse

Write an algorithm that takes a word, and outputs the word backwards.

Solution

function wordReverse(word) {
 var newWord = “”;
 for (var i = word.length - 1; i >= 0; i--) {
  newWord += word[i];
 }
 return newWord;
}

Find Number

Given an array of numbers, write an algorithm that takes a number and returns true, if that number exists in the array. Extra credit, if it also returns the index where the number is found. The array is:

var numArray = [2, 49, 65, 7, 88, 23, 44, 33];

Solution

function findNumber(num) {
 for (var i = 0; i < numArray.length; i++) {
  if (num == numArray[i]) {
   return true; // or return num + “ is found at index ” + i;
  }
 }
 return false;
}

Rocky Fight Rounds

Apollo Creed has 100 health points. Write an algorithm that takes a number, which represents Rocky’s punch power, and returns how many rounds it would take for Rocky to defeat Apollo.

Solution

function rockyPunchApollo(punchPower) {
 var apolloLife = 100;
 var roundCount = 0;
 while (apolloLife > 0) {
  apolloLife -= punchPower;
  roundCount++;
 }
 return roundCount;
}

3/26/2024 - - Object Methods

Overview

An object method is simply a function that you can access within an object.

1. Syntax

We've briefly looked at these, but we haven't gotten specific about what they are, and how to build them. Below is the basic syntax for a method within an object.

var person = {
 firstName: "John",
 lastName : "Doe",
 id : 5566,
 fullName : function() {
  return this.firstName + " " + this.lastName;
 }
};

2. Calling a Method

Call the above "fullName" method:

person.fullName();

3. Look Familiar?

Remember when I said everything in JavaScript is an object? You’ve been working with methods on Strings, or Arrays, etc. In essence, those String variables you create, are instances of a greater String object that has several methods baked into it. Like:

String.length;

String.toUpperCase();

String.toLowerCase();

String.replace();

4. Weekly Challenges

A really good way to practice solving algorithms, and to get really comfortable with JS built in methods is to try and create your own function that imitates something already done with a built in method.

For instance, we know String.length returns the length of a string. How would you write an algorithm that does the same thing? Here's one way:

function getLength(string) {
 var stringLength = 0;
 var stringArray = string.split(“”); // Turns string into an array
  while (stringArray[0] != undefined) { // While array has an item
   stringArray.pop(); // Removes an array item per loop
   stringLength++; // increases length variable by 1
 }
 return stringLength; // Returns the final count
}
